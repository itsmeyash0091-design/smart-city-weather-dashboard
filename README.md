# 🌦️ Smart City Weather Dashboard

An intelligent weather dashboard that goes beyond temperature to provide **actionable insights** like "What to Wear" and real-time weather recommendations.

---

## 🚀 Features

* 🌍 Search weather by city
* 📍 Auto-detect location using Geolocation API
* 📅 5-day weather forecast
* 👕 Smart "What to Wear" suggestions
* 🌈 Dynamic UI (background changes with temperature)
* 🌦️ Weather icons based on conditions

---

## 🧠 Smart Insights

This app doesn't just show weather — it helps you **decide what to do**:

* 🌧️ Rain > 20% → *"Carry an umbrella!"*
* 🥶 Cold weather → *"Wear a jacket!"*
* 🌡️ Hot weather → *"Stay hydrated & wear light clothes!"*

## 🎨 UI/UX Ideas

* 🔵 Cold weather → Blue theme
* 🟠 Hot weather → Orange theme
* 🌧️ Rain → Dark cloudy theme

---

## ⚙️ How It Works

1. User opens app
2. App detects location using Geolocation API
3. Fetches weather data from OpenWeatherMap API
4. Displays:

   * Current weather
   * 5-day forecast
   * Smart suggestions
